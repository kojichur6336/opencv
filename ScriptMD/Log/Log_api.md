# Log 功能接口文档

---
## 1. `Log.open()`

### 1.1 接口说明

`Log.open()` 用于 **初始化并打开日志文件**。当日志系统未初始化或打开失败时，将返回 `false`。

---

### 1.2 参数说明

无需参数

---

### 1.3 返回值说明

| 返回值 | 类型    | 中文说明 | 示例 |
|--------|---------|----------|------|
| 返回值 | boolean | 打开结果 | `true` |

---

### 1.4 Lua 调用示例

```lua
local ok = Log.open()
if ok then
    Log.print('日志系统已成功打开')
else
    Log.print('日志系统打开失败')
end
```

---
## 2. `Log.close()`

### 1.1 接口说明

`Log.close()` 用于 **关闭日志功能**，关闭后不再输出日志。

---

### 1.2 参数说明

无需参数

---

### 1.3 返回值说明

该接口 **无返回值**。

---

### 1.4 Lua 调用示例

```lua
Log.close()
```

---
## 3. `Log.print()`

### 1.1 接口说明

`Log.print(...)` 用于 **打印日志信息**，支持传入多个参数并依次输出。

---

### 1.2 参数说明

| 参数位置 | 类型 | 是否必选 | 中文说明                 | 示例 |
|----------|------|----------|--------------------------|------|
| 第 1 个  | any  | 是       | 日志内容的第 1 个参数     | `'tag'` |
| 第 2 个  | any  | 否       | 日志内容的第 2 个参数     | `count` |
| 第 3 个  | any  | 否       | 日志内容的第 3 个参数     | `err` |
| 第 N 个  | any  | 否       | 日志内容的后续参数（变参） | `value` |

---

### 1.3 返回值说明

该接口 **无返回值**。

---

### 1.4 Lua 调用示例

```lua
local count = 3
local err = nil
Log.print('Count', count, 'Error', err)
```
